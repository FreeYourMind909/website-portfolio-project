# Website Portfolio Project

## Project Overview

This repository contains a portfolio website project for **IBM 3202: Market Analysis and Control** at Cal Poly Pomona. The project showcases analytical skills and coursework completed throughout the semester using R, Quarto, and business intelligence tools.

## Project Purpose

The website serves as a professional portfolio demonstrating competency in:
- **Data Wrangling**: Using tidyverse (dplyr, tidyr) for data cleaning and transformation
- **Data Visualization**: Creating insights with ggplot2 and the grammar of graphics
- **Reproducible Reports**: Building professional documents with Quarto/R Markdown
- **Interactive Presentations**: Creating engaging slide presentations with Reveal.js
- **Marketing Analytics**: Analyzing and visualizing marketing data
- **Dashboard Development**: Summarizing data insights in interactive dashboards
- **Web Analytics**: Evaluating website performance with Google Analytics

## Project Structure

The portfolio website includes the following sections:

### 1. **Home Page**
- Professional introduction
- Academic background and career interests
- Overview of key skills developed
- Navigation to different portfolio sections

### 2. **Portfolio Projects**
- Data Visualization Projects
- Data Wrangling Exercises
- Advanced Data Wrangling Concepts
- Each with explanations of methodology and insights

### 3. **Dashboard**
- Interactive visualizations of marketing data
- Performance metrics and trends
- Key business indicators

### 4. **Reveal.js Presentation**
- Interactive slideshow of project work
- Data analysis examples
- Analytical insights and storytelling

### 5. **Website Analytics**
- Traffic statistics and visitor engagement
- User behavior patterns
- Performance metrics
